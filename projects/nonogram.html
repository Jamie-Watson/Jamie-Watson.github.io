Nonogram Generator

May 7th, 2024

Personal Project Python Based

One of my personal projects emerged from my frustration with the excessive ads in many nonogram mobile games. In response, I decided to create my own simple, ad-free version using Python. This project is still in its early stages but is a good visualizer for randomly generated nonogram boards. While the current implementation is relatively straightforward, itâ€™s also efficient, and I see plenty of potential for future development and enhancements. The foundation of the project is a class called "Board," which takes a single parameter, n, as the side length. This class is responsible for generating a random n by n square board. Presently there is only a feature for square boards as this is what is most commonly seen using nonograms. Alongside this, I've included a function to print the board, ensuring that the rows and columns are neatly aligned which is typical for a nonogram board. Although the project is in its beginning stages, I am working on improvements to make it more engaging and polished. This is a simple program which has allowed me to apply and refine my programming skills while working on something I'm passionate about.

# Function Return Type Purpose
1 __init__(self, size) void Creates an object of the Board class with a side length of "size"
2 get_side_values(self) tuple of (list of int, list of int) Creates a row and column list of the values on the side of the board which indicate how many filled in squares exist
3 __str__(self) string Returns the string form of the board object (the board without the side values)
4 get_row_buffer(self) int Helper method which returns the number of spaces needed to align the rows
5 get_column_max_size(self) int Helper method which returns the number of spaces needed to align the columns
6 print_table(self) void Prints and aligns the board
