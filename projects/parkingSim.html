Java Parking Lot Simulator

March 22nd, 2024

School Project Java Based

Methodology

This Java program is a 13 class comprehensive simulation designed to optimize parking lot management by determining the ideal number of parking spaces required in a parking lot based on car arrival rates. It was developed as a project for my ITI1121 (Introduction to Computing II) course. The program assumes a single entrance and exit point, with cars being processed one at a time. The simulation is based on the number of cars arriving at the parking lot per hour.

The program calculates the likelihood of any car leaving based on the time it has spent in the parking lot. The longer a car has been parked, the greater the probability that it will leave. This feature ensures that the simulation realistically mimics the flow of vehicles in and out of a parking lot. The program continuously processes incoming cars, allowing them to enter the parking lot as long as there is available space. If the lot is full, incoming cars must wait in a queue until a spot becomes available.

The goal of the simulation is to determine the optimal number of parking spots needed to minimize congestion and waiting times. The program begins by assuming a single parking space and simulates 10 separate 24-hour periods, averaging the number of cars waiting to enter the lot. During these trials, if the average length of the incoming queue exceeds five cars, the program increases the number of simulated parking spaces by one and reruns the trials. This iterative process continues until the program finds an optimal number of spaces that minimizes the average queue length.

Class Purpose
1 CapacityOptimizer Main class that simulates and contains the main method
2 Car Class that creates the car objects
3 LinkedQueue Implements the queue data structure using a singly linked list
4 List Implements the list interface
5 ParkingLot Class that creates the parking lot object and contains relevant helper methods
6 Queue Implements the queue interface
7 RandomGenerator Generates the license plate for the cars and calculates whether a car leaves
8 Rational Class that creates rational numbers and contains relevant helper methods
9 Simulator Class that simulates one day of the optimization cycle
10 SinglyLinkedList Implements the singly linked list data structure
11 Spot Class that creates the parking spot objects
12 StudentInfo Displays information about the author (me)
13 TriangularDistribution Class that calculates probability density

Each trial's results are outputted, providing detailed data on the parking lot's performance under various conditions. By analyzing this data, the program can recommend the most efficient number of parking spaces needed to accommodate the predicted car arrival rates while minimizing the inconvenience of waiting in line. This simulation not only helps in designing more efficient parking lots but also in understanding traffic patterns and optimizing resource allocation in real-world scenarios.
