Pen-Pal: Reading Aid

March 19th, 2023

Personal Project Java Based

This is a program I developed during my initial foray into learning Java, aimed at enhancing accessibility for individuals with reading difficulties. The program is fully designed using Java, encompassing both frontend and backend elements, and leverages Java APIs to provide a seamless user experience. The primary goal of this project is to make reading more accessible and enjoyable by offering a range of customizable features. One of the key functionalities is the ability for users to adjust the font size, which can significantly improve readability for those with visual impairments or dyslexia. Additionally, the program incorporates a text-to-speech feature that reads the text aloud, providing an auditory option for users who may struggle with traditional reading. This feature is particularly useful for those with severe reading difficulties or who prefer auditory learning.

Function Purpose
1 Adjustable Font Size Allows users to enlarge text, improving readability for those with visual impairments or dyslexia.
2 Text-to-Speech API Provides auditory reading options, aiding users who struggle with traditional reading or prefer auditory learning.
3 Integrated Dictionary (using Jsoup) Enables quick access to word definitions, enhancing comprehension and reducing interruptions.
4 Adjustable Background and Foreground Colors Customizes the visual appearance to reduce eye strain and improve focus, especially for users sensitive to certain color contrasts.
5 Box Layout Manager Ensures consistent layout and easy navigation regardless of window resizing, maintaining usability for all users.

Additional Features

To further assist users, the program includes an integrated dictionary feature. This was implemented using the Jsoup web scraping library, allowing users to quickly look up definitions and meanings of words directly alongside the text they are reading. This immediate access to definitions can greatly enhance comprehension and reduce interruptions caused by needing to look up unfamiliar words. Moreover, the program allows users to adjust the background and foreground colors of the text, catering to individual preferences and needs. This customization can alleviate eye strain and improve focus, especially for users who are sensitive to certain color contrasts. The user interface is designed with flexibility in mind, utilizing the BoxLayout manager to maintain proportions and layout integrity when the user resizes the window. This ensures that the program remains visually coherent and user-friendly, regardless of screen size adjustments.

Project Goals

Overall, this program is a comprehensive tool aimed at enhancing the reading experience for individuals with various needs and preferences. While it started as a simple Java learning project, it has evolved into a functional and valuable accessibility tool. I plan to continue refining and expanding this program, exploring new features and enhancements to better serve its users.
